Privacy Policy for STL (School Tool Launcher)
Effective Date: January 2025
A1dos Creations is committed to protecting the privacy of STL (School Tool Launcher) users. This document explains the data practices and privacy commitments for STL.

Data Collection STL does not collect, transmit, or store any user data on external servers. All data associated with STL remains on the user’s device and within the Chrome browser environment. No personal identifiable information (PII), browsing history, or private information is accessed or stored by STL.

Permissions STL uses the following permission strictly for its functionality: 1. Storage Permission: This is used to store user preferences and optional data such as theme settings, favorited buttons, tasks, and Pomodoro Timer configurations. All stored data is kept locally on the user’s Chrome profile.

Data Usage The data stored by STL is used exclusively to provide the following features:
- Customizable themes for user preference.
- Quick access to favorited buttons.
- Task management and organization.
- A functional Pomodoro Timer for productivity.

STL does not sell, share, or distribute any user data to third parties. All operations occur client-side, ensuring full user control over their information.
Third-Party Integrations STL does not include any third-party trackers, analytics, or integrations. The only external resource used is Google Fonts to enhance the visual design. These fonts are loaded directly from Google’s servers and do not involve any data collection or interaction with STL.

Internet Access STL does not require or utilize internet access for its core features. Aside from fetching fonts from Google Fonts, STL is designed to operate entirely offline and does not connect to any external servers or APIs.

User Responsibilities To ensure the security and proper functioning of STL, users are advised to:
- Keep their Chrome browser updated to the latest version.
- Manage their extension permissions periodically.
- Report any issues or concerns directly to A1dos Creations.

Contact Information If you have questions, concerns, or need assistance related to STL’s privacy practices, you can reach A1dos Creations through the following channels:
Email: rbentertainmentinfo@gmail.com
Official Website: https://www.a1dos-creations.com
GitHub Repository: https://github.com/A1dos-Creations/STL

Commitment to Privacy A1dos Creations values user trust and is dedicated to maintaining STL as a privacy-respecting, user-focused tool. This privacy policy will be reviewed regularly and updated as necessary to reflect any changes in STL’s functionality or practices.
